Que es TailwindCSS?

Es un framework altamente customizable para el desarrollo rápido de interfaces personalizadas.

Como usar TailwindCSS?

Con TailwindCSS das estilo a tus elementos usando clases pre-definidas directamente en el HTML sin escribir (a veces poco) CSS directamente.

Beneficios

No pierdes tiempo y energía definiendo nombres para tus clases

El CSS nunca crece.

Hacer cambios rápidos y de forma segura

Instalación

CDN

https://unpkg.com/tailwindcss@^1.0/dist/tailwind.min.css

NPM

npm install tailwindcss autoprefixer
